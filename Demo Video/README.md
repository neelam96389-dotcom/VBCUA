# 🎥 Demo Video — VBCUA

This folder contains the link to the recorded demonstration of the **Voice-Based Concept Understanding Analyser (VBCUA)** in action.

🔗 **Watch the Demo (Google Drive):** 
https://drive.google.com/file/d/11JqN0BQf5R0UjD1v8_fPccHPhP9HivKN/view?usp=sharing

---

## What the video shows

1. **Home Screen** — audio upload box and Concept Reference selection
2. **Uploading Audio** — selecting a concept and uploading a spoken explanation
3. **Analysis in Progress** — running transcription, semantic scoring, and audio feature extraction
4. **Results Screen** — transcript, semantic similarity score, filler word ratio, pause ratio, confidence (energy), waveform visualization, and final understanding score
5. **PDF Report** — downloading and viewing the generated PDF report

---

## Notes

- Google Drive sharing is set to **"Anyone with the link — Viewer"** so the video can be opened without requesting access.
- This video demonstrates the application running locally (`streamlit run app.py`) — no public deployment was used, as documented in the Project Executable Files (Folder 6).
